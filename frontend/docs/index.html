API Documentation

Complete reference for Travel Planner API

Authentication

All API requests require JWT authentication via Bearer token.

Authorization: Bearer YOUR_TOKEN_HERE

POST /api/auth/register

Register a new user account

Request Body:

{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "password123",
  "contact": "+1-555-0000"
}

Response:

{
  "userId": "abc123",
  "token": "eyJhbGciOiJIUzI1...",
  "name": "John Doe",
  "email": "john@example.com"
}

POST /api/trips

Create a new trip

Request Body:

{
  "title": "Tokyo Adventure",
  "destination": "Tokyo, Japan",
  "startDate": "2024-06-01",
  "endDate": "2024-06-10",
  "totalBudget": 3000
}

GET /api/trips/:id/budget

Calculate trip budget breakdown

Response:

{
  "activitiesTotal": 170,
  "accommodationTotal": 1350,
  "expensesTotal": 700,
  "totalBudget": 2220
}

POST /api/ai/itinerary-suggest

Get AI-powered itinerary suggestions

Request Body:

{
  "destination": "Paris, France",
  "dates": {
    "start": "2024-08-15",
    "end": "2024-08-22"
  },
  "preferences": "Art museums and French cuisine"
}
